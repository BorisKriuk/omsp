# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [1.0.0] — 2026-04-06

### Added
- Initial public release.
- Core protocol implementation with full documentation.
- Docker support with `docker-compose.yml` for one-command setup.
- Comprehensive test suite with benchmark results.
- `README.md` with setup instructions, usage guide, and architecture overview.
- `CONTRIBUTING.md` with contribution guidelines and branch conventions.
- `SECURITY.md` with responsible disclosure policy.
- MIT License.

### Security
- Environment-based configuration — no hardcoded secrets.
- Input validation and sanitization across all endpoints.
- Rate limiting enabled by default.
